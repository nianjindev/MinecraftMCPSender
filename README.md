# Minecraft MCP
Connects to `http://localhost:8080` and sends data such as interactions with blocks, entities, chat messages, as well as environmental changes.

### MCP Server
Run the MCP server with [MinecraftNarratorMCP](https://github.com/1J6K21/MinecraftNarratorMCP)

### Features
Can detect player actions:
- Eating
- Crafting
- Mining blocks
- Placing blocks
- Taking damage
- Hitting Entities
- Changes in your biome
- Changes in day/night
- Chat messages
- System messages

# Installation/Usage
Grab the latest release .jar file and add it to your mod folder.

Make sure you're using the latest Fabric version on minecraft 1.21.10.

May add to Modrinth/Curseforge soon.
